Schema Reference

x-substrukt fields

  • title — Display name for the content type
  • slug — URL-safe identifier (auto-generated from title if omitted)
  • storage — directory (one file per entry) or single-file (all entries in one JSON file)
  • kind — collection (multiple entries) or single (one entry, e.g. site settings)

Supported property types

  • string — text input. Add "format": "textarea", "markdown", "upload", or "reference"
  • number / integer — numeric input with optional min/max
  • boolean — checkbox
  • object — nested fieldset
  • array — repeatable items (define "items" schema)
  • enum — dropdown select (add "enum": ["a", "b"] to a string)

Extensions

  • "x-control": "textarea" — render string as multi-line textarea
  • "x-substrukt-reference": {"schema": "slug"} — reference to another content type
